# HCN's Collections of Nonpareils (or Nonsense)

[![PyPI version](https://img.shields.io/pypi/v/hcn.svg)](https://pypi.org/project/hcn/)
[![Python Version](https://img.shields.io/badge/python-3%2B-blue)](https://www.python.org/downloads/)
[![License](https://img.shields.io/github/license/huchaoning/hcn?color=blue)](https://github.com/huchaoning/hcn/blob/main/LICENSE)


**HCN** is a collection of personal Python utilities.  


## Key Features

- **I/O Helpers** – Simplified file handling.  
- **Parallelization** – Easy-to-use wrappers for multiprocessing and job distribution.  
- **Plotting Presets** – Ready‑to‑use Matplotlib style sheets (including a MATLAB‑like style).  
- **Cryptography Tools** – Convenient interfaces for hashing, encryption, etc.  
- **Email Utilities** – Lightweight email sending functions.  
- **Miscellaneous** – Caching, CPU ID detection, and other small helpers.
